Lokale Planung für deutsche Haushalte

Finanzielle Lebensplanung, ohne Familiendaten wegzugeben.

LiF hilft dabei, Cashflow, Depot, Immobilien, Darlehen, Renten, Schenkungen, Kinder und Ruhestandsszenarien über Jahrzehnte zu modellieren. Die App läuft lokal, startet mit synthetischen Demodaten und macht die Zahlen nachvollziehbar.

Was es ist

Zukunftsplanung, kein Budget-Klon.

Budget-Tools sind stark im Heute. LiF ist für das nächste Jahr, die kommenden fünf Jahre, Kinder, die größer werden, Entscheidungen zum Darlehen, Depot-Entnahmen, Schenkungen, Immobilien und Ruhestand. Aus einem Haushaltsmodell entstehen Monats- und Jahresprojektionen, die man prüfen kann, statt einer Blackbox zu vertrauen.

Was modelliert wird

Gebaut für echte Haushaltsentscheidungen.

01

Familie und Einkommen

Erwachsene, Kinder, Kindergeld-artige Einnahmen, Gehälter, Bonuszahlungen, RSUs, Gehaltsänderungen und Lebensereignisse.

02

Konten und Depots

Girokonten, Tagesgeld, Depots, Kindervermögen, Anleihen, Ausschüttungen, geplante Käufe und Umbuchungen zwischen Konten.

03

Darlehen und Immobilien

Hypotheken, Zinsbindungen, Anschlussfinanzierung, private Darlehen, Immobilienbesitz und Übertragungsplanung.

04

Ruhestand und Szenarien

Gesetzliche Rente als Planungsmodell, private Renten, FIRE-Cash- Ziele, Szenario-Kopien, Reviews und Datenqualitätschecks.

Local-first Versprechen

Deine Daten bleiben dort, wo LiF läuft.

  • Kein gehosteter Sync und kein nötiger Cloud-Account.
  • Keine Telemetrie in der App.
  • Keine Drittanbieter-Finanz-API als Standard.
  • Synthetische Demo-Daten zum Ausprobieren vor echten Zahlen.

Schnellstart

Demo lokal starten.

LiF ist eine Django-App. Repository klonen, gesperrte Abhängigkeiten installieren, Migrationen ausführen, Demo-Daten laden und im Browser öffnen. 

git clone https://github.com/lif-planner/lif.git
cd lif
pipenv install
pipenv run python manage.py migrate
pipenv run python manage.py seed_demo
pipenv run python manage.py runserver 127.0.0.1:8000
