Plan your household's money, and your path to FIRE, on your own machine.

LiF is a self-hosted personal-finance and retirement-projection tool built around how money actually works for a family in Germany — Kindergeld, statutory pension assumptions, depot holdings, mortgages, and long-horizon projections you can audit line by line.

View on GitHub Read the docs

Local-first, always private

Runs on your own machine or a server you control. Your account balances, salary, and family data never leave it — no cloud account, no telemetry, no third party in the loop.

Built for how Germany actually works

Kindergeld, statutory pension-style retirement assumptions, Vorabpauschale/Teilfreistellung-aware depot modeling, and mortgage refinance assumptions — not a generic spreadsheet template.

Every number is explainable

Monthly and yearly audit pages trace every projected balance back to the opening balance and the line items that changed it. Nothing is a black box.

What it models

Household & family

Adults, children, Kindergeld-style child income, child milestones, and salary changes over time.

Accounts & depots

Manual or CSV-imported accounts, depot holdings with per-holding ETF distribution rates, and transfer rules between them.

Debt & mortgages

Interest/principal split repayment, fixed-interest periods, and refinance assumptions.

Retirement & equity

German statutory pension-style retirement income and RSU/equity grant vesting income.

Long-term projections

Monthly-granularity calculations rolled up into yearly reports, for as many decades out as you need.

Scenarios & analytics

Clone a household into a what-if scenario and compare outcomes side by side.

Imports

Preview-first CSV imports for accounts and depot holdings, with a dry-run table before anything is applied.

Data quality & audits

Built-in readiness checks and full monthly/yearly audit pages for every projected figure.

Get started

LiF runs locally with Python and Django. No account, no signup, no external service required.

git clone https://github.com/yogitea/LiF.git
cd LiF
pipenv install
pipenv run python manage.py migrate
pipenv run python manage.py seed_demo
pipenv run python manage.py runserver 127.0.0.1:8000

Then open http://127.0.0.1:8000/. See the README for container and always-on deployment options.
